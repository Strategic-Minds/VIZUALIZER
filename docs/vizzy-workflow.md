# Vizzy Workflow

Vizual-X is a white, artsy, voice-or-text floor visualizer for Vercel.

## Customer Flow

1. Upload a floor photo.
2. Say or type a command to Vizzy.
3. Vizzy searches for style ideas and shows three rectangular options.
4. The user clicks an option or says which option they like.
5. Vizzy outlines the floor.
6. Vizzy renders the chosen epoxy style onto the detected floor area.
7. The app plays a small completion chime and lets the user download the preview.

## Production Gates

- Add OpenAI env vars in Vercel.
- Confirm real product names and colors.
- Add customer consent and photo-retention policy.
- Add quote/CRM handoff later.
- Human-review generated renders before final quote.
